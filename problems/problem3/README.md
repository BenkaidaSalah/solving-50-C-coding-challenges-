# Problem 3: Sum of Each Column in Matrix

## 🧠 Description
This program generates a 3x3 matrix filled with random numbers and calculates the sum of each column.

## 📥 Input
No user input is required. The matrix is generated randomly.

## 📤 Output
- A 3x3 matrix of random numbers
- The sum of each column

## 🛠️ Concepts Used
- 2D Arrays
- Functions
- Loops
- Column operations

## 🎯 Goal
Practice working with columns in a matrix and improve problem-solving skills.

## 📌 Example Output
12 45 78
34 22 11
90 10 55

Column 1 Sum = 136
Column 2 Sum = 77
Column 3 Sum = 144
