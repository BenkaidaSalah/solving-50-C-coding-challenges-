# Problem 4: Store Column Sums in Array

## 🧠 Description
This program generates a 3x3 matrix with random numbers, calculates the sum of each column, and stores the results in a separate array.

## 📥 Input
No user input is required. The matrix is generated randomly.

## 📤 Output
- A 3x3 matrix of random numbers
- The sum of each column stored and displayed from an array

## 🛠️ Concepts Used
- 2D Arrays
- Functions
- Loops
- Arrays (1D & 2D)
- Data processing

## 🎯 Goal
Practice working with matrices and storing computed results in a separate array.

## 📌 Example Output
10 20 30
40 50 60
70 80 90

Column 1 Sum = 120
Column 2 Sum = 150
Column 3 Sum = 180
