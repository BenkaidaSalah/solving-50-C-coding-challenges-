# Problem 2: Sum of Each Row in Matrix

## 🧠 Description
This program generates a 3x3 matrix filled with random numbers and calculates the sum of each row.

## 📥 Input
No user input is required. The matrix is generated randomly.

## 📤 Output
- A 3x3 matrix of random numbers
- The sum of each row

## 🛠️ Concepts Used
- 2D Arrays
- Functions
- Loops
- Random number generation

## 🎯 Goal
Practice working with matrices and improve problem-solving using functions.

## 📌 Example Output
12 45 78
34 22 11
90 10 55

Row 1 Sum = 135
Row 2 Sum = 67
Row 3 Sum = 155
